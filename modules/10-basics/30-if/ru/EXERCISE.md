Вашей задачей в этом задании будет написать функцию `check_number()`, которая число и возвращает число с описанием числа. Функция принимает два числа и возвращает числа:

* Если число больше нуля, то возвращает `positive`,
* Если меньше нуля то `negative`,
* Если равно нулю то `zero`

Пример:

```ruby
# Примеры вызова:
puts check_number(10)   # => "positive"
puts check_number(-5)   # => "negative"
puts check_number(0)    # => "zero"
```

Функция должна **возвращать** строку.
